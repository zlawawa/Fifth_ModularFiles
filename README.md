# Fifth_ModularFiles
C++. Лаба №5 (6 вариант). Строки.

## №1 - String46
Дана строка, состоящая из русских слов, разделенных пробелами (одним или несколькими). Найти длину самого длинного слова.

## №2 - Str18
Дана строка, содержащая путь к файлу в формате **ОС MS DOS**. Нужно преобразовать путь в формат **Unix**, заменив все обратные слеши **(\\)** на прямые **(/)**.

## №3 - Str23
Дана строка. Реализовать метод шифрации **"Наложение гаммы"**: **"новый_код = старый_код XOR гамма"**. Гамма находится в диапазоне от 127 до 255. Расшифровать строку при заданной гамме.

## №4 - Str44
Дана строка. Вывести символ, который встречается наибольшее число раз, и количество этих символов. Если есть несколько символов с одинаковым количеством, вывести последний из них.

## №5 - Str31
Дано число в **10-ой** системе счисления. Перевести это число в **18-ю** систему счисления.

## №6 - 3
Напишите **программу-телеграф**, которая принимает от пользователя сообщение и выводит его на экран в виде последовательности точек и тире. Использовать **Азбуку Морзе** для букв русского и латинского алфавитов.

## №7 - Five6
В текстовом файле c именем **FN1** даны примеры домашней работы девятиклассника. в формате: **<операнд><знак><операнд>=<ответ>**. Операнды являются не более чем трехзначными шестнадцатеричными числами, в качестве знаков операций используются сложить **'+'** и умножить **'*'**. Нужно проверить домашнюю работу девятиклассника, учитывая, что ответ также должен быть шестнадцатеричным числом. Результат проверки записывается в выходной файл с именем **FN2**, после неверно решенных примеров ставится знак **'!'**.
